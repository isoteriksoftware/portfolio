Real-Time Virtual Voice Assistant Based Yanshee Humanoid Robot for Point to Point Navigation



In this project, a real-time virtual voice assistant was implemented on the Yanshee humanoid robot for point-to-point navigation. The robot is utilized for point-topoint navigation based on voice commands so as to address the problems associated with efficiency of human assistants working 24 hours daily while executing certain processes and navigating same path repeatedly.
